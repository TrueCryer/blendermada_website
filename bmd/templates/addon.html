How to use:

  1. Install it like typical add-on for Blender (here is a link)
  2. Turn on the add-on
  3. Find add-on panel at Properties window inside Materials tab
  4. Press Update button to connect to server
  5. To enable/disable preview just press preview button. When preview is active, you can drag it with middle mouse button
  6. Select material and press Import

Client is still under active development. If you find a bug, of have another issue, you can find this project on GitHub or write me directly - ozzyrov@gmail.com
